# 배경 꾸미기
## 이미지를 배경에 사용하기
- background-image: 이미지 사용
- background-repeat: 이미지가 배경 크기보다 작을 떄 이미지를 반복해서 배경을 어떻게 채울지 결정
- background-size: 이미지 크기 
  - contain: 이미지의 가로/세로 중 긴 쪽을 배경에 맞추고 짧은 쪽은 여백으로 남김(center, no-repeat 시)
  - cover: 배경에 이미지 크기를 맞춤

## 배경에 그라데이션 넣기
