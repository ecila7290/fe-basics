|---|---|---|

## id 값을 사용해 요소로 이동
href 값에 #id값을 넣어준다.
외부 사이트의 특정 요소로 이동하는 것도 가능.

## address
태그 안에 주소와 연락처 정보를 넣을 수 있다.
- href=tel: 모바일 등에서는 전화로 연결 가능
- href=mailto: 클릭하면 이메일 보내는 도구 열림
