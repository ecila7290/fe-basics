# 종류와 중요도에 따른 태그

## 강조용 태그
- b 태그 vs strong 태그
b 태그는 글자를 굵게만, strong 태그는 그 내용이 중요하다는 뜻도 담겨 있음

- i 태그 vs em 태그
i 태는 글자를 기울게만 표시, em 태그는 강조(emphasis)

옛날에는 b태그나 i태그와 같은 태그가 디자인적인 기능도 했지만 요즘은 그런 작업을 전부 css에서 하기 때문에 디자인적인 것을 위해 해당 태그를 사용할 일은 없다.
